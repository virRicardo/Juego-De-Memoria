# JuegoMemoJW

## Descripción del Juego

JuegoMemoJW es un juego de memoria clásico donde el jugador debe encontrar pares de cartas idénticas. El tablero consta de 16 cartas (8 pares) que se voltean al hacer clic. El objetivo es emparejar todas las cartas antes de que se agote el tiempo.

## Características

- **Temporizador**: 30 segundos para completar el juego.
- **Contadores**: Muestra el número de movimientos realizados y aciertos obtenidos.
- **Efectos de sonido**: Sonidos para clics, pares correctos, incorrectos, victoria y derrota.
- **Animaciones**: Confeti al ganar el juego.
- **Reinicio**: Botón para reiniciar el juego en cualquier momento.

## Tecnologías Utilizadas

- **HTML**: Estructura del juego.
- **CSS**: Estilos y diseño visual.
- **JavaScript**: Lógica del juego, temporizador, sonidos y animaciones.

## Cómo Jugar

1. Abre `index.html` en un navegador web.
2. Haz clic en las cartas para voltearlas.
3. Encuentra pares coincidentes.
4. Gana antes de que el tiempo se acabe.

## Requisitos

- Navegador web moderno con soporte para HTML5, CSS3 y JavaScript.
